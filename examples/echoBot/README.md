# Echo Bot

## Setup

Add your **api_id**, **api_hash**, **bot_token** in *settings.json*.

## Requirements

This example doesn't require additional packages.

## Running

Enter in *echoBot* folder and run:

```shell
python main.py
```

## Input

Write some text messages to your *echoBot*. Example:

```
>> You: Hey Bot!
>> You: Hi!
```

## Output

Bot response:

```
>> Bot: Hey Bot!
>> Bot: Welcome!
```
